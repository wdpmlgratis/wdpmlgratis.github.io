# Halo, Ini Testing MDX!

Ini adalah postingan untuk ngetes apakah komponen link kita berhasil.

### 1. Link Standar (Markdown)
Ini link biasa: [Google](https://google.com) (Harusnya terbuka di tab yang sama).

### 2. Link Custom (Tanpa External)
Ini pake komponen: <Link href="https://github.com">GitHub</Link> (Harusnya tetap tab yang sama).

### 3. Link Custom (Pake External)
Ini yang sakti: <Link href="https://astro.build" external>Website Astro</Link> 🚀
(Harusnya terbuka di **TAB BARU** karena ada kata kunci `external`).
